# 📌 Todo App

![License](https://img.shields.io/badge/License-MIT-blue)
![Made with](https://img.shields.io/badge/Made%20with-HTML%20%7C%20CSS%20%7C%20JS-6a0dad)
![Status](https://img.shields.io/badge/Status-Active-brightgreen)

A modern, gradient‑styled to‑do list app built with **HTML**, **CSS**, and **vanilla JavaScript**. Add tasks, mark them complete, filter by status, and clear completed items — all within a clean, intuitive interface designed for simplicity and productivity.

## ✨ Features
- ➕ Add new tasks  
- ✔️ Mark tasks as completed  
- 🔄 Toggle tasks on click  
- 🎯 Filter tasks (All / Active / Completed)  
- 🧹 Clear completed tasks  
- 🎨 Modern gradient UI  
- 📱 Responsive design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Google Fonts (Jost)  
- Font Awesome Icons
